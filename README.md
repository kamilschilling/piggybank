Co zrobić aby uruchomić projekt? 

git clone repo 

stworzyć/[wybrać istniejący] virtualenv 

instalacja pakietów potrzebnych do uruchomienia w requirements.txt:

[pip install -r requirements.txt] i powinno sie wszystko zainstalowac co trzeba

przejsc do backend/piggybank: cd piggybank

uruchomić serwer: python manage.py runserver
